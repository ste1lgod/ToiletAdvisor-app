# ToiletAdvisor — Презентация проекта

---

## Что это такое

**ToiletAdvisor** — мобильное веб-приложение для поиска общественных туалетов в Ташкенте.

Пользователь открывает сайт на телефоне, видит интерактивную карту города
и все отмеченные туалеты поблизости. Можно найти ближайший, посмотреть
подробности, почитать отзывы других людей и оставить свой.

Приложение работает **без скачивания из магазина** — открывается в браузере,
но при желании устанавливается на экран телефона и выглядит как обычное приложение.

---

## Что умеет приложение

### Для пользователя
- 🗺️ **Интерактивная карта** — все туалеты на карте Ташкента, цвет маркера показывает открыт/закрыт
- 📍 **Найти ближайший** — одна кнопка находит ближайший туалет по GPS
- 🔍 **Поиск и фильтры** — поиск по названию, фильтры по категориям (бесплатно, мыло, бумага, доступность, тахаратхана)
- 📋 **Карточка туалета** — адрес, статус, цена, характеристики, отзывы с оценками
- ⭐ **Отзывы** — зарегистрированные пользователи оставляют оценки и комментарии
- ❤️ **Избранное** — сохранение любимых туалетов с синхронизацией между устройствами
- 🌙 **Тёмная тема** — переключение светлой и тёмной темы
- 🌐 **Три языка** — русский, узбекский, английский
- 📶 **Офлайн-режим** — при отсутствии интернета показывает сохранённые данные

### Для администратора
- ➕ **Добавление туалетов** — удобный пошаговый визард: выбор места на карте, заполнение данных, предпросмотр
- 🗂️ **Модерация** — просмотр и удаление отзывов пользователей
- 📋 **Журнал действий** — полная история всех действий в системе с фильтрами
- 📊 **Статистика** — сводка по точкам и пользователям

## Технические характеристики

| Параметр | Значение |
|----------|---------|
| Тип | Прогрессивное веб-приложение (PWA) |
| Платформа | Любой браузер, iOS и Android |
| Установка | Без App Store и Google Play — прямо из браузера |
| Хостинг | GitHub Pages (бесплатно, надёжно) |
| База данных | Google Firebase Firestore (облако) |
| Карта | Яндекс.Карты |
| Адреса | OpenStreetMap / Nominatim |
| Деплой | Автоматический при каждом обновлении кода |
| Серверные расходы | $0 (бесплатный план Firebase и GitHub Pages) |

## Безопасность

- Пароли пользователей хранятся в зашифрованном виде (SHA-256 — односторонний алгоритм, расшифровать невозможно)
- Права доступа настроены в базе данных — обычные пользователи не могут удалять чужие данные
- Приватные ключи не хранятся в открытом коде

## Что было сделано в ходе последнего аудита (июль 2026)

Проведена полная проверка всех данных и кода. Найдено и устранено **12 проблем**:

- Исправлена ошибка из-за которой страница статистики падала при загрузке
- Устранено двойное геокодирование которое вызывало лишние запросы к серверу
- Исправлены неверные адреса в избранном у пользователей
- Улучшена скорость загрузки: отзывы и избранное открываются мгновенно из кэша
- Очищена база данных: удалены 12 тестовых дублирующихся аккаунтов
- Устранена ошибка при сохранении ника пользователя
- Улучшена надёжность отображения имён авторов отзывов
